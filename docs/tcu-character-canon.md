# TCU Character Canon

## Core Characters
- Chef Maurice / Trading Chef
- Chef Goldie
- Wickie
- Louie the Liquidity Chef
- Burn Alarm
- Profit Plate
- Melissa Mayhem
- Grandma Market

## Expanded Roster
- Candle Kid
- Melody Mayhem
- Rico Rhythm
- Penny / Angel Stacks
- Nana Value
- Mr. Stocks

## Visible Character Layer
Characters must be visible in the interface, not only stored in docs.
The terminal should show character coaching cards, warnings, and pathway guidance.

## Mapping Rules
- Missing bias → Grandma Market
- Missing liquidity → Louie the Liquidity Chef
- Missing confirmation → Wickie
- Missing burn point → Burn Alarm
- Missing targets → Profit Plate
- FOMO / revenge / chasing → Melissa Mayhem
- XAUUSD / Gold → Chef Goldie
- Clean setup → Chef Maurice / Trading Chef
- Beginner candle confusion → Candle Kid
- Session rhythm → Melody Mayhem
- Momentum / delivery → Rico Rhythm
- Small account / lot size → Penny / Angel Stacks
- AOI / value-zone patience → Nana Value
- Stocks / ownership = Mr. Stocks

## Character Tone
Use coach-forward language:
- Chef recommends
- Setup flagged
- Practice anyway
- Learning entry saved
- Kitchen Ticket
- Burn Point
- Tables Served
- Prep Zone
- Protect the Plate
- Candle Proof
- Market Kitchen
- Trading Chef University

## Canon Notes
Characters are teaching agents, not market voices. Keep them focused on process, risk awareness, and the learning journey.
