# TCU Visual Reference Map

## Identity Overview
The TCU Market Kitchen Terminal must feel like a premium trading education dashboard with a bold chef/kitchen motif.

Key visual traits:
- premium black/navy dashboard base
- purple MS glow accents
- gold roadmap highlights
- green BUY and red SELL actions
- low-opacity TCU Market Kitchen watermark
- TradingView/HugoX/Maven-inspired panel structure
- clearly labeled Practice / Simulation mode

## Core UI Sections
- Top Bar: strong brand identity, mode indicators, grade, and session details
- Chart Kitchen: static practice chart, upload controls, timeframe buttons, concept pills, drawing toolbar, legend, watermark
- Execution Panel: BUY/SELL, Market/Pending toggle, lot size, risk mode, stop loss, take profit, trailing stop, roadmap selectors, grade, ticket actions
- Bottom Tabs: practice positions, prep orders, served plates, chart read, trade plan, journal, mistake review, concepts, mission board
- Character Layer: visible coaching crew, warnings, and lesson guidance

## Chart Kitchen Reference Elements
- candlesticks
- grid background
- price axis
- time axis
- support/resistance lines
- gold Liquidity Sweep marker
- purple AOI / Prep Zone overlay
- blue FVG / imbalance zone
- red Burn Point line
- green Tables Served target lines
- NY Breakfast Rush label
- Bias label
- TCU Market Kitchen watermark
- disclaimer text: `Practice chart for education · Not live market data`

## Brand & Style
- MS purple crown/logo should appear in the Top Bar and other key interface anchors
- Market Marina and Scott-King Coast world references should be visible in the terminal
- Use premium type, bold uppercase labels, and chef/kitchen terminology consistently
- Keep the design clearly educational and practice-focused

## Notes
This reference map is a guide for the UI upgrade phase. Do not make visual changes until the required assets and canon docs are in place.
