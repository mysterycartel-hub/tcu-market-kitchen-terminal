## Core Rule
Do not assume a new build starts from zero.  
Check this source first.  
Reuse existing canon and assets before creating anything new.

## Intent Rule
Every request starts as one of these:
Research, Audit, Compare, Store, Monitor, or Build.

Only enter Build Mode when Maurice says:
Build it, Execute, Create it, Launch it, or Prototype it.

- no asset exists
- the current asset is unclear
- higher resolution is needed
