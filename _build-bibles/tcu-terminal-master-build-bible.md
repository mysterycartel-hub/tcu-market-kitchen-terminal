# TCU Market Kitchen Terminal — Master Build Bible

## Product Identity
- Product Name: TCU Market Kitchen Terminal
- Parent Brand: MysterMyself
- World: Scott-King Coast
- District: Market Marina
- Product Line: Trading Chef University / TCU

## Core Product Definition
A free-to-use educational trading terminal where users can explore charts, practice BUY/SELL planning, upload screenshots, mark setups, receive TCU roadmap grading, get coached by Trading Chef Universe characters, save Kitchen Tickets, journal mistakes, and turn lessons into future Remotion video content.

This is not a broker.
This is not a signal app.
This is not financial advice.
This is a practice and education terminal.

## Core Behavior Rule
Never block the student.
Always:
- reveal risk
- warn clearly
- teach the lesson
- save the learning entry
- let the student decide

## Locked Roadmap
Bias → Liquidity → AOI → Delivery → Confirmation → Entry → Targets → Management

## Roadmap Meanings
- Bias = Kitchen Direction
- Liquidity = Flow
- AOI = Prep Zone
- Delivery = How Price Moves
- Confirmation = Candle Proof
- Entry = The Pass
- Targets = Tables Served
- Management = Protect the Plate

## Locked Trading Language
BUY and SELL buttons must stay visible.
Do not replace them with only “Bullish Scenario” or “Bearish Scenario.”
Correct structure:
- BUY button = opens BUY Kitchen Ticket
- SELL button = opens SELL Kitchen Ticket
The platform teaches the setup logic behind the buttons.
